# ASP.NET.8.Identity
Sample Implement of new Identity using Minimal ASP.NET API 8 and FastEndPoint and REPR pattern With Result pattern.
and used different mapping methods like manual mapping and [Mapperly](https://github.com/riok/mapperly).
## 2 Endpoint: 
- ```api/v1/register/Consumer``` to register as consumer, used Mapperly.
- ```api/v1/register``` to register application used, used manual mapping.
